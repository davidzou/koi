### 介绍
旨在快速创建gradle plugin项目工程，提供快速稳定的创建方式

### 创建方式

1. shell 文件创建
    
    ```
    bash koi_create.sh 
    ```
2. jar 方式
    
    ```
    java -jar tarball/koi-1.0.0.jar -n plugindemo
    ```
